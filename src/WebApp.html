DARToSheeToDART

신속하고 투명한 전자공시시스템, DARToSheeToDART

1) 작업 폴더 지정

작업 루트 폴더 URL 또는 ID를 입력하세요. 하위에 source/worksheet/backupsheet/target이 자동 생성됩니다.

2) DSD 업로드 후 워크시트 생성

DSD 파일을 source 폴더에 업로드하고 worksheet/backupsheet를 자동 생성합니다.

작업 시트 링크를 저장하십시오.

작업 시트는 고객님이 지정한 작업 폴더 내에서 찾을 수 있습니다.

3) Sheet → DSD 변환 실행

생성된 워크시트 URL 또는 ID를 입력하고 변환을 실행하세요. 결과는 target 폴더에 저장됩니다.
